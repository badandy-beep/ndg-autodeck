V23 CHANGES FROM V22 (OPTIMIZATIONS PER ANALYSIS)
═══════════════════════════════════════════════════════════════════════════════

1. MOBILE VIEW OPTIMIZATION:
   - Increased minimum font sizes:
     * body: 0.9rem → 1.1rem min
     * bodyS: 0.9rem → 1rem min
     * hero: 2.5rem → 3rem min
     * main: 2rem → 2.5rem min
   - Larger image sizes:
     * imgHero: 250px → 300px
     * imgLarge: 300px → 350px
     * imgMedium: 260px → 300px
     * imgSmall: 200px → 240px
     * imgSeated: 180px → 220px
   - Increased padding (0 1rem → 1rem 1.5rem) for breathing room
   - Container now uses justifyContent: 'flex-start' (was 'center')
   - Added overflowY: 'auto' for scrolling on long content slides

2. TEXT CRAWL ADJUSTMENTS:
   - Slower scroll speed (0.22 → 0.18) for reading pace
   - Reading pace adjusted (200 WPM → 180 WPM)
   - Dynamic height calculation - crawl continues until content disappears
   - Full transition (transitionAt: 95% → 100%)
   - Word-by-word highlighting ADDED to mobile (was desktop-only)
   - 500ms pause before advancing to deck phase

3. AESTHETICS ENHANCEMENTS:
   - Brighter gold color (#fbbf24 → #ffd700)
   - Added subtle text shadows for depth:
     * mainT/slideT/svcT: 0 2px 4px rgba(0,0,0,0.3)
     * body/pre/attr: 0 1px 2px rgba(0,0,0,0.2)
   - Larger crawl fonts (desktop):
     * crawl-h: 2.6rem → 3rem
     * crawl-p: 1.7rem → 2rem
     * crawl-large: 2.1rem → 2.5rem
   - Larger crawl fonts (mobile):
     * header: 1.5rem → 1.8rem
     * paragraphs: 1rem → 1.1rem
     * large: 1.2rem → 1.4rem

4. IMAGE BIGGER IMPACT (Desktop):
   - imgHero: 400px → 450px
   - imgLarge: 450px → 500px
   - imgMedium: 380px → 420px
   - imgSmall: 300px → 340px
   - imgSeated: 280px → 320px

5. CREDENTIALS SLIDE FIX (White Box Sizing):
   - Changed grid to flexbox with wrap for auto-sizing
   - Reduced logo box padding (1rem → 0.5rem desktop, 0.5rem → 0.4rem mobile)
   - Boxes now hug logos more closely, eliminating "twice as big" issue
   - Added minWidth/maxWidth constraints for consistent sizing

6. GLOBALS.CSS ENHANCEMENTS:
   - Added mobile base font-size: 16px for visibility
   - Added .crawl-content { width: 95% !important } for mobile

DEPLOYMENT INSTRUCTIONS
═══════════════════════════════════════════════════════════════════════════════

1. Extract ZIP to your project directory
2. Run: npm install
3. Run: npm run dev (development) or npm run build && npm start (production)
4. For Vercel: Push to GitHub or drag-drop to Vercel dashboard

Note: Next.js 14.0.0 has a security vulnerability. Consider upgrading to a 
patched version after deployment.
