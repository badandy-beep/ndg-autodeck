# NDG AutoDeck

NDG AutoDeck is an advanced automation utility designed to streamline deck management tasks. It leverages cutting-edge algorithms to optimize workflows, saving both time and effort for users.

## Features
- **Automated Deck Creation**: Generate decks quickly with minimal input.
- **Customizable Layouts**: Tailor layouts to match user preferences.
- **Advanced Sorting Options**: Organize decks efficiently based on various criteria.
- **User-Friendly Interface**: Simple and intuitive design for ease of use.

## Installation

To install NDG AutoDeck, clone the repository and set up the environment as follows:

```bash
git clone https://github.com/badandy-beep/ndg-autodeck.git
cd ndg-autodeck
pip install -r requirements.txt
```

## Usage

After installation, you can use NDG AutoDeck by running the main script:

```bash
python main.py
```

Follow the on-screen instructions to create and manage your decks.
